## ✨ 技術實作

### 錨點導航
所有章節都有 ID 錨點：

```html
<div id="quick-start">快速開始</div>
<div id="modes">三種模式</div>
<div id="concepts">關鍵概念</div>
<div id="faq">常見問題</div>
```

### URL 結構
```
docs.html           → 文檔首頁
docs.html#quick-start → 快速開始章節
docs.html#modes     → 三種模式章節
docs.html#concepts  → 關鍵概念章節
docs.html#faq       → 常見問題章節
```
