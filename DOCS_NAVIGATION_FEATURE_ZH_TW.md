# 文檔導航功能 - 使用說明

## ✅ 新增功能

在應用程式的頂部導航欄右側，新增了 **📖 Docs** 按鈕，提供快速存取所有等待時間指標文檔。

---

## 📍 位置

```
頂部導航欄
┌─────────────────────────────────────────────────────────┐
│ [FlowSim] [主題] [狀態] [時鐘] [📖 Docs] [選單]         │
└─────────────────────────────────────────────────────────┘
                                    ↑
                                  在這裡
```

---

## 🎯 使用方式

### 步驟 1：點擊 Docs 按鈕
在頂部導航欄右側找到 **📖 Docs** 按鈕並點擊。

### 步驟 2：選擇文檔
下拉選單會顯示四個選項：

```
┌─────────────────────────────────────┐
│ 等待時間指標文檔                    │
│ 選擇適合你的指南                    │
├─────────────────────────────────────┤
│ ⚡ 快速參考                         │
│    5 分鐘快速上手三種模式           │
├─────────────────────────────────────┤
│ 📖 完整指南                         │
│    詳細說明與實際案例               │
├─────────────────────────────────────┤
│ 🗂️ 文檔索引                        │
│    所有資源導航                     │
├─────────────────────────────────────┤
│ 🔬 深入分析                         │
│    雙指標詳細解析 (EN/简)           │
└─────────────────────────────────────┘
```

### 步驟 3：在新分頁開啟
點擊任何選項，文檔會在新分頁中開啟，不會影響你當前的工作。

## 🎨 設計特點

### 響應式設計
- **桌面版**：顯示完整的 "Docs" 文字
- **平板/手機**：僅顯示圖標節省空間

### 視覺設計
- **漸層標題**：藍色到綠色的漸層效果
- **圖標標識**：每個選項都有獨特的 emoji
- **懸停效果**：滑鼠懸停時背景變亮
- **分組顯示**：繁體文檔在上，其他語言在下

### 互動體驗
- 點擊按鈕開啟選單
- 點擊選單外關閉選單
- 點擊文檔項目在新分頁開啟
- 不中斷當前工作流程

## 💡 使用建議

### 第一次使用系統
```
1. 點擊 📖 Docs
2. 選擇「⚡ 快速參考」
3. 花 5 分鐘閱讀
4. 回到系統開始使用
```

### 遇到具體問題
```
1. 點擊 📖 Docs
2. 選擇「📖 完整指南」
3. 查看「常見問題」部分
4. 或查看相關使用場景
```

### 尋找特定資訊
```
1. 點擊 📖 Docs
2. 選擇「🗂️ 文檔索引」
3. 按場景或角色找到相關文檔
4. 點擊連結查看
```

### 深入技術理解
```
1. 點擊 📖 Docs
2. 選擇「🔬 深入分析」
3. 閱讀技術詳解
4. 理解計算邏輯
```

## 🎓 常見問題

### Q: 文檔在哪裡開啟？
**A**: 新的瀏覽器分頁，不會影響當前工作。

### Q: 可以離線閱讀嗎？
**A**: 可以，文檔是本地 Markdown 檔案。

### Q: 為什麼有些文檔是英文？
**A**: 技術深度文檔保留英文/簡體，適合開發者和分析師。

### Q: 手機上能用嗎？
**A**: 手機螢幕較小，文檔按鈕可能隱藏。建議在桌面/平板使用。
